Detailed:
SnippetController
    GET    /api/v1/snippets
        Get all snippets available for a user
        Response: a map containing the id of each snippet associated with the user's permission
            {
                snippet1: owner,
                snippet2: shared,
                ...
            }
        If the user is not found, returns 404
    PUT    /api/v1/snippets/{snippetId}?language={language}
        Register a snippet into App's database. It also adds owner permission to the current user.
        This endpoint is meant to be used by Runner after creating a snippet, using the same JWT used for the creation request
        No request body is required as the userId is taken from the JWT and the language is a PathVariable
    DELETE /api/v1/snippets/{snippetId}
        Delete a snippet
    PUT    /api/v1/snippets/{snippetId}/permission
        Share a snippet with another user
        Request:
            {
                userId: {userId}
            }
    DELETE /api/v1/snippets/{snippetId}/permission
        Remove permission for another user
    DELETE /api/v1/users
        Delete a user from all databases
    GET    /api/v1/snippets/{snippetId}/tests
        Get all tests for a snippet
        Response: a list containing all tests IDs
            {
                [testId]
            }
    POST   /api/v1/snippets/{snippetId}/tests
        Create a test
        Request:
            {
                input: [String]
                expected: {String}
            }
        Response
            {
                testId: {testId}
            }
    PUT    /api/v1/snippets/{snippetId}/tests/{testId}
        Start execution of a test
    DELETE /api/v1/snippets/{snippetId}/tests/{testId}
        Delete a test
    POST   /api/v1/snippets/{snippetId}/run?version={version}
        Start execution of a snippet or provide input
        Version is not required
        Request (not required for starting):
            {
                input: {String}
            }
        Response:
            {
                status: COMPLETED/OUTPUT/WAITING/ERROR,
                message: String
            }
    DELETE /api/v1/snippets/{snippetId}/run
        Cancel execution of a snippet
    PUT    /api/v1/rules
        Modify task rules
        Request:
            {
                task: {formatting/linting}
                language: {language}
                rules: {
                    rule1: {var1}
                    rule2: {val2}
                    ...
                }
            }
    GET    /api/v1/rules?task={task}&language={language}
        Get all user's rules for a specific task and language
        Response:
            {
                rule1: {val1}
                rule2: {val2}
                ...
            }

Additional endpoints:
    These endpoints are meant for develop and debugging only.
    These endpoints should be removed from production as well as their associated logic

            {
                task: "test",
                userId: "userId",

        If successful, Runner should print:
            ###########################################################
            # TESTING TESTING TESTING TESTING TESTING TESTING TESTING #
            ###########################################################
            #
            # {snippetId}
            #
            ###########################################################
            # TESTING TESTING TESTING TESTING TESTING TESTING TESTING #
            ###########################################################
